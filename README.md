# Habit Tracker

A single-page React habit tracker with weekly checkmarks, current streaks, week navigation, and localStorage persistence.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL Vite prints in the terminal, usually `http://localhost:5173`.

## Build

```bash
npm run build
```
